Projects So Far...

  • Submitted: DataSift: Selective Data Expansion for Model Performance
    [View Paper]

    Highlights of this work:
    • Developed a novel framework, DataSift, to acquire selective data points for fair and accurate ML training
    • Integrated a Multi-Armed Bandit algorithm into the data acquisition process
    • Implemented influence functions to improve framework effectiveness and efficiency
    • Experimental results outperform existing baselines
  • Submitted: PipeLens: Identifying Interventions for Optimizing Data Science Pipelines

    Highlights of this work:
    • PipeLens treats the entire pipeline—data cleaning, transformations, feature engineering, and models—as the object of diagnosis and repair.
    • Instead of guessing or correlating failures, PipeLens applies causally verified interventions (parameter changes, module insertion, deletion, or reordering) to identify true root causes.
    • Uses historical pipeline executions to learn a proxy utility model, guiding automatic repairs toward changes that are most likely to improve performance.
    • Automatically reasons over dataset properties to align failing data with previously successful distributions.
  • Published: Sentinel: Explanations for machine learning pipelines under data drift. [View Paper]

    Highlights of this work:
    • Quantifying and assessing the robustness of data preprocessing pipelines under data shift
    • Explaining pipeline robustness and model vulnerability for end-users
    • Estimating performance degradation due to future data shifts
